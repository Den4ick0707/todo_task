# Todo Task
Реалізовано програму для введення задач
Програма складається з двох частин - *Frontend* і *Backend*

## Щоб запустити програму потрібно:

1. Мати підключення до інтернету
2. Потрібно щоб були встановленні *Docker* та *Node.js*
2. Завантажити архів та розпакувати його 
3. Виконати в терміналі таку команду в директорії де файл README.md:
```
    cd ./todo_task_backend
```

4. Потім створити файл .env та ввести туди це:
```
    PORT=3000
    MONGO_URI=mongodb://db:27017/db
    DB_NAME=db
    NODE_ENV=development
    CORS_ORIGINS=http://localhost:3000
    LOG_LEVEL=debug
```
5. Потім виконати команду: 
```
    docker-compose up --build
```

6. Коли все успішно запустилося переходимо до директорії Frontend:
    cd ../todo_task_frontend
    
7. Потім виконуємо команду:
```
    npm install
```
    та 
```
    gulp
```

8. Коли все вдало запустилося потрібно перейти за посиланням:
```
    http://localhost:8080/
```

## Програму реалізованно за допомогою:

### Backend
- Фреймворк - Express.js
- База данних - MongoDb
- Документація API - Swagger(http://localhost:8080/api-docs)
- Використанно Docker для контейнеризації та Docker compose для запуску та налагдження декількох контейцнерів

### Frontend
- HTML,
- SCSS(препроцесор для CSS)
- JavaScript
- Використанно збірник Gulp
- Розроблялося все за мотодологією BEM
- Використано плагіни для Gulp які мініфікують та зменшують розмір коду
- Налаштовано LiveReload та Watcher
